liunx命令

##ubuntu镜像下载

http://cn.ubuntu.com/download/
  • 终端字体大小：Ctrl shift +；Ctrl -
  • ls：显示当前路径目录文件及文件夹名字
  • ls -l 显示列表
  • ls -l -h 显示文件大小
  • ls -a 显示隐藏和不隐藏的所有文件
  • pwd 查看当前路径
  • touch：创建一个文件 touch xiaoxin 可以带后缀也可以不带后缀 liunx没有后缀概念
  • mkdir 创建一个文件夹
  • clear 清屏
  • cat ，more查看文件
  • history 历史命令
  • ！2074 执行2074对应的命令
  • rm 删除命令

  • 重定向 ls > xx.txt:创建xx.txt，删除原来内容再把把结果放到xx.txt里面

  • 追加 ls >> xx.txt :创建xx.txt，不删除原来内容把结果放进去

  • ls -alh /bin | more
  • table 自动补全，两次table 显示当前所有文件
  • . 当前路径 ；.. 当前路径的上一个路径
  • cd - 上一次所在的目录； cd ~ 家目录
  • 当你写了一行命令 但是你不想执行这个命令，Ctrl + c 就是不执行的意思
  • whoami 查看当前用户名 exit 退出
  • ctrl shift t 打开两个标签； alt 1； alt 2 快速切换标签
  • cal 查看日历;cal -y 2017 查看2017年日历
  • date 查看当前时间
  • ps -aux;top ;htop;kill 进程名；kill -9 进程名 强制结束
  • vim编辑器： i esc ： w保存 q 退出，wq保存退出
  • 命令模式：yy复制，4yy 复制4行，pp粘贴，dd剪切，2dd剪切2行
  •         D 剪切光标后面的到行末
  •         u 撤销操作
  •         ctrl+r 反撤销
  •      kjhl 上下左右
  •      HML  快速定位光标位置 上中下
  •      Ctrl+f向下翻一页 Ctrl+b向上翻一页，ctrl+d 向下翻半页，ctrl+u向上翻半页
  •      20G 快速定位第二十行
  •      G 回到行末
  •      gg 回到第一行
  •      w向后跳一个单词长度
  •      b先前跳一个单词长度
